# PullTheTimer Pro Plus 3000 — site & mises à jour

Dépôt **public** de [PullTheTimer Pro Plus 3000](https://sandrophoto.github.io/pullthetimer-pro-plus-3000/),
le minuteur de la barre de menu qu'on règle en tirant une bille.

Il ne contient que ce qui doit être public :

- `docs/` — le site, servi par GitHub Pages
- `appcast.xml` — le flux Sparkle interrogé par les apps installées
- `releases/appcast.xml` — l'appcast généré à la publication

Le code source de l'app est privé. Les binaires sont dans les
[Releases](https://github.com/sandrophoto/pullthetimer-pro-plus-3000/releases).
